# elm-hn

Hacker news client developed in elm

## Develop

### Requirements

- [node & npm](https://nodejs.org/en/download/)

### Getting started

``` sh
git clone git@github.com:Uhsac/elm-hn.git
./script/bootstrap
./script/start_dev
```

### Deploy

``` sh
./script/deploy
```
